## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### run storybook

```sh
npm run storybook
```

### build storybook

```sh
npm run build-storybook
```

### run the storybook build

```sh
npx http-server path-to-storybook-static
```
